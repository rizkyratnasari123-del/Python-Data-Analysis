# Python Data Analysis – Coffee Sales Portfolio

## Project Overview

This project performs **exploratory data analysis (EDA)** on coffee sales data to uncover business insights, trends, and customer behavior using Python.

## Objectives

* Clean and prepare raw sales data
* Analyze sales trends and product performance
* Visualize insights using Python libraries
* Provide data-driven business conclusions

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

## Key Insights

* Identified top-selling coffee products
* Observed sales trends over time
* Detected customer purchasing patterns
* Provided recommendations for business improvement
